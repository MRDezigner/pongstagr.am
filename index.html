Pre-requisites

In order to use this plugin, you will need your user-id and access token to access Instagram API. more here

So, set your user-id & access_token
  • If you have zero idea what your user id is, you may head to this link: http://bit.ly/WIz7xe
  • If you have zero idea what your access token is, you may head to this link: http://bit.ly/TLUOjB make sure you follow the instructions on the "How do I get my client id?" link.
  • and we're set!

Pongstagr.am jQuery plugin v2.0

It's on alpha, but it's available use it in any way you can, licensed under MIT. Let me know what you think! hit me up! @pongstr

Example: Recent Media

$('div#selector').pongstgrm({
  user_id      : insta_id   ,  /*! Your Instagram User-ID          */
  access_token : insta_token,  /*! Your Instagram Access Token     */
  show         : 'recent'   ,  /*! Type of media to be returned    */
  media_count  : 8             /*! Number of media to be displayed */
});
