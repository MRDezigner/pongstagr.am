Documentation

Pongstagr.am is a jQuery plugin that lets you display your Instagram media to your website using Bootstrap Front-end styles and modal-plugin.

Requirements

For the most part, Instagramâ€™s API only requires the use of a client_id. A client_id simply associates your server, script, or program with a specific application. However, some requests require authentication - specifically requests made on behalf of a user. Authenticated requests require an access_token. These tokens are unique to a user and should be stored securely. Access tokens may expire at any time in the future more here.

  1. Instagram Account   [user id & access token]
  2. jQuery 1.10.x
  3. Bootstrap v3.x or head over here to use the plugin-only version

Usage

  1. Head over to Jelled Web Design Blog and Look up your Instagram ID and ACCESS TOKEN:.

  2. Initialise stylesheets, jQuery Library, Bootstrap JS libray before calling on the plugin like the example below: 


<!-- Initialize stylesheets to header -->
<link rel="stylesheet" href="/path/to/css/bootstrap.css">
<link rel="stylesheet" href="/path/to/css/pongstagr.am.css>

<!-- Initialize Javascripts to footer -->
<script href="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.js""></script>
<script href="/path/to/js/boostrap.js"></script>
<script href="/path/to/js/pongstagr.am.js"></script>

Call on the plugin like this:

By default the plugin displays eight (8) most recent media. 
$(document).ready(function () {

  $('#selector').pongstgrm({
    accessId:     'YOUR-ACCESS-ID',
    accessToken:  'YOUR-ACCESS-TOKEN'
  });

});

Options:

Override default options. 
{
  show:       string    // show your recent media, likes, feed or any tag you'd like to see
  count:      8         // number of media you'd like to display, default is 8. Instagram's maximum is 40
  likes:      boolean   // true: show likes count, false: doesn't show it
  effects:    string    // add your css3 animation/transition for thumbnails
  comments:   boolean   // true: show comments count, false: doesn't show it
  timestamp:  boolean   // true: show media timestamp, false: doesn't show it
}

More Options

Override default options. 
{
  preload:     "spinner", // custom selector for preload spinner
  button:      "btn btn-success", // custom selector for load more button
  buttontext:  "Load more", // custom text for load more
  column:      "col-xs-6 col-sm-3 col-md-3 col-lg-3", // custom columns for thumbnail
  likeicon:    "glyphicon glyphicon-heart", // custom likes icon
  commenticon: "glyphicon glyphicon-comment" // custom comment icon
}

Examples

Limits: Be nice. If you're sending too many requests too quickly, Instagram will send back a 503 error code (server unavailable). You are limited to 5000 requests per hour per access_token or client_id overall. Practically, this means you should (when possible) authenticate users so that limits are well outside the reach of a given user.

Profile

Display your Instagram basic information.

Recent Media

Display your most recent media.

User Liked

Display your most recent liked media.

User Feed

Display the most recent media from the people you follow .

Tag

Display the most recent media available using a tag .
